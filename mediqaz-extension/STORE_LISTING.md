# MediQaz — Chrome Web Store Listing

## Название
MediQaz — AI Ассистент Врача

## Краткое описание (132 символа)
AI-ассистент для врачей: запись приёма → транскрипция → медкарта → автозаполнение МИС. Для системы здравоохранения Казахстана.

## Полное описание

### 🩺 MediQaz — ваш AI-ассистент на каждом приёме

MediQaz — это расширение Chrome, которое превращает голосовую запись врачебного приёма в структурированную медицинскую карту за секунды.

**Как это работает:**
1. 🎤 Нажмите «Начать приём» — расширение записывает разговор
2. 📝 В реальном времени транскрибирует речь (русский/казахский)
3. 🤖 AI генерирует медицинскую карту из 7 секций
4. 🏥 Одним кликом заполняет формы МИС (Damumed, e-MIS)

**Ключевые возможности:**

✅ **Живая транскрипция** — Deepgram Nova-2 с диаризацией (🔵 Врач / 🟢 Пациент)
✅ **AI медкарта** — жалобы, анамнез, диагноз (МКБ-10), назначения, рекомендации
✅ **Linked Evidence** — нажмите ▶ рядом с секцией и услышьте точный момент записи
✅ **Magic Edit** — уточните секцию голосом: «Добавь температуру 38.5»
✅ **Автозаполнение МИС** — инжекция данных в формы Damumed и e-MIS
✅ **Q&A по приёму** — задайте вопрос: «Пациент упоминал аллергию?»
✅ **PDF экспорт** — скачайте медкарту в формате PDF
✅ **Голосовые команды** — скажите «Медиказ, запомни аллергию на пенициллин»

**Для кого:**
- Терапевты, педиатры, кардиологи, хирурги, ЛОР, неврологи
- Врачи поликлиник и стационаров Казахстана
- Специалисты, работающие в Damumed, e-MIS, и других МИС

**Безопасность:**
- 🔒 Все данные хранятся ТОЛЬКО на вашем устройстве
- 🔒 Нет облачных серверов — нет утечек
- 🔒 API ключи зашифрованы в chrome.storage
- 🔒 Соответствие Закону РК «О персональных данных»

**Горячие клавиши:**
- Ctrl+Shift+M — начать/остановить запись
- Ctrl+Shift+G — сгенерировать медкарту
- Ctrl+Shift+P — скачать PDF
- Ctrl+Shift+Q — задать вопрос по приёму

**Требования:**
- API ключ Deepgram (бесплатный план: 1000 минут/месяц)
- API ключ Groq (бесплатный план доступен)
