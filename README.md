# MediQaz — AI-ассистент врача

AI-медицинский скрайб для врачей Казахстана. Записывает приём, транскрибирует речь, автоматически заполняет формы Дамумед.

## Архитектура

```
Браузер (PWA)
    ↓ WebSocket (аудио-чанки)
Node.js + Express
    ↓ REST API
Deepgram Nova-2 (STT + диаризация спикеров)
    ↓ транскрипция
Groq — Llama 3.3 70B (генерация JSON)
    ↓ структурированные данные
SQLite (Better-sqlite3)
```

## Поддерживаемые формы

| Код   | Название                        |
|-------|---------------------------------|
| 052/у | Амбулаторный приём              |
| 035/у | Лист временной нетрудоспособности |
| 130/у | Рецептурный бланк               |

## Быстрый старт

```bash
# 1. Скопируйте шаблон переменных окружения
cp .env.example .env

# 2. Заполните .env своими ключами (см. раздел "API-ключи")

# 3. Установите зависимости
npm install

# 4. Запустите в dev-режиме
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`.

## Запуск через Docker

```bash
# Заполните .env, затем:
docker-compose up --build
```

Приложение будет доступно по адресу `http://localhost:80`.

## API-ключи

| Переменная        | Где получить                                       |
|-------------------|----------------------------------------------------|
| DEEPGRAM_API_KEY  | https://console.deepgram.com/ (бесплатный тариф)   |
| GROQ_API_KEY      | https://console.groq.com/ (бесплатный тариф)       |
| JWT_SECRET        | Сгенерируйте: `openssl rand -base64 32`            |

## Ограничения

- Deepgram Nova-2 лучше работает с чистым аудио (без фонового шума)
- Groq Free Tier: лимит запросов ~30/мин — поэтому цикл обработки = 20 сек
- Диаризация (Спикер 0 = Врач, Спикер 1 = Пациент) — эвристика, может ошибаться
